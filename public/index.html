常见问题

我的 API Key 安全吗？

本站不存储你的 API Key。请求结束后立即从内存中清除，且不会写入任何日志。

黑盒检测能覆盖大部分常见的欺诈手段，但无法保证 100% 准确。建议结合长期使用体感综合判断。

检测一次需要消耗多少 Token？

完整检测约消耗 1k–5k Token，具体取决于所选模型。
